# 🎮 罗小黑战记·森林双跳游戏

## 📌 项目介绍

这是一个使用 **C++ 和 EasyX 图形库** 开发的经典双跳游戏，融合了《罗小黑战记》治愈系森林世界观的设计元素。

**核心玩法**：通过左右移动和双跳机制，踩在不断下落的平台上，不断向上攀爬，避免掉出屏幕。
